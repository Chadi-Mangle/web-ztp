# Fonctionnalités principales

Voici les fonctionnalités disponibles sur l’interface web et dans le backend :

## Interface Web

- Gestion des **templates ZTP**
- Création, édition et suppression de **périphériques (devices)**
- Utilisation d'un template pour configurer un device
- Visualisation des **dashboards**
- Authentification utilisateur (login/signup)
